# Setting Up Google Sheets Template

## For the Buyer (After Purchase):

The Google Sheets calculator needs to be set up as a "Make a Copy" template.

### Step 1: Create the Calculator
1. Go to sheets.google.com
2. Create new spreadsheet
3. Set up 5 tabs as outlined in google-sheets-template.txt
4. Add all formulas and formatting

### Step 2: Share as Template
1. File → Share → Share with others
2. Set to "Anyone with the link can view"
3. Copy the link
4. Change the URL from:
   `https://docs.google.com/spreadsheets/d/[ID]/edit?usp=sharing`
   to:
   `https://docs.google.com/spreadsheets/d/[ID]/copy`

### Step 3: Update thanks.html
Replace the placeholder link in thanks.html with the actual /copy link
