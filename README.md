## 📊 Calculator Features

### Inputs
- Property Address
- Square Footage
- After Repair Value (ARV)
- Rehab Level (Light/Medium/Heavy)
- ARV Discount % (default 70%)
- Assignment Fee Target
- Contingency % (default 15%)

### Outputs
- Estimated Rehab Cost (auto-calculated)
- MAO (Maximum Allowable Offer)
- Suggested Initial Offer (85% of MAO)
- Suggested Max Offer (95% of MAO)
- Buyer All-In Cost
- Buyer Potential Profit
- Deal Grade (A/B/C/D)

### Phoenix-Specific
- +5% labor cost adjustment for Phoenix market
- Links to Redfin, Zillow, Maricopa Assessor
- Local investor group references

## 🚀 Remaining Setup (5 minutes)

### Step 1: Create Google Sheet
1. Go to https://sheets.new
2. Create 5 tabs matching the structure in `google-sheets-template.txt`
3. Add formulas and yellow-highlight input cells
4. Format with $ and % where appropriate

### Step 2: Share Template
1. File → Share → Anyone with link can VIEW
2. Copy the link
3. Change `/edit?usp=sharing` to `/copy` at the end
4. Example: `https://docs.google.com/spreadsheets/d/[ID]/copy`

### Step 3: Update Thank You Page
1. Edit `thanks.html`
2. Replace placeholder link with your actual /copy link
3. Commit and push to GitHub

### Step 4: Test Purchase Flow
1. Visit https://ryanmartincre.github.io/phoenix-wholesale-calc/
2. Click Buy Now
3. Complete PayPal test purchase
4. Verify redirect to thank you page
5. Confirm Google Sheets link works
